## Complexity Analysis:

### Time Complexity patterns:
```
Single loop:        O(n)
Nested loops (2):   O(n²)
Nested loops (3):   O(n³)
Try all subsets:    O(2ⁿ)
Try all permutations: O(n!)
```

### Space Complexity :
Brute force usually has **O(1) space** because:
- Only uses variables
- No extra data structures
- In-place operations

---

## Key Takeaways

1. **Always works** - Guaranteed solution (if exists)
2. **Easy to implement** - Good starting point
3. **Foundation for optimization** - Understand problem first
4. **Interview safety net** - Better than no solution
5. **Time vs Space tradeoff** - Usually O(1) space, high time
6. **Not always bad** - Perfect for small inputs
7. **Learning tool** - Builds problem-solving intuition
